# Turtle Crossing

## Overview

A Frogger-inspired Turtle Crossing game developed in Python using the Turtle graphics library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Player-controlled turtle movement
- Randomly generated traffic
- Collision detection
- Level progression
- Increasing car speed with each level

---

## Technologies

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)
