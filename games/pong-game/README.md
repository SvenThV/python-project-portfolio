# 🏓 Pong Game

A classic Pong game built with **Python** and the **Turtle** graphics module as part of the **100 Days of Code: The Complete Python Pro Bootcamp**.

## 📖 Overview

This project recreates the iconic Pong arcade game. Two players control paddles on opposite sides of the screen and try to prevent the ball from passing their side. The game demonstrates object-oriented programming by separating the game logic into multiple classes.

## ✨ Features

- Two-player gameplay
- Ball movement with realistic bouncing
- Paddle controls
- Collision detection with paddles and walls
- Automatic ball speed adjustment
- Score tracking
- Clean object-oriented project structure

## 🛠️ Technologies

- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)

## 📂 Project Structure

```
pong-game/
│── main.py          # Main game loop
│── paddle.py        # Paddle class
│── ball.py          # Ball movement and collision logic
│── scoreboard.py    # Scoreboard management
```

## 🎮 Controls

| Player | Move Up | Move Down |
|---------|---------|-----------|
| Left Paddle | W | S |
| Right Paddle | ↑ | ↓ |

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/SvenThV/python-project-portfolio.git
```

2. Navigate to the project folder:

```bash
cd python-project-portfolio/games/pong-game
```

3. Run the game:

```bash
python main.py
```
