# Pong Game

## Overview

A classic Pong game developed in Python using the Turtle graphics library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Two-player gameplay
- Paddle movement
- Ball movement and bouncing
- Collision detection
- Score tracking

---

## Technologies

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)
