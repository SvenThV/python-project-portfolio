# Snake Game

## Overview

A classic Snake game developed in Python using the Turtle graphics library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Snake movement
- Food spawning
- Collision detection
- Score tracking
- Increasing game difficulty

---

## Technologies

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)
