# Password Manager

## Overview

A desktop Password Manager developed in Python using the Tkinter GUI library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Generate secure random passwords
- Copy generated passwords to the clipboard
- Save website credentials to a JSON file
- Search for saved credentials
- User-friendly graphical interface

---

## Technologies

- Python
- Tkinter
- JSON
- Pyperclip
- Object-Oriented Programming (OOP)
