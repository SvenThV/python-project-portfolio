# Pomodoro Timer

## Overview

A desktop Pomodoro Timer developed in Python using the Tkinter GUI library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Pomodoro work and break timer
- Automatic work/break session management
- Long break after every four work sessions
- Session progress tracking with check marks
- Simple graphical user interface

---

## Technologies

- Python
- Tkinter
