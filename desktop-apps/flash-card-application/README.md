# Flash Card App

## Overview

A desktop Flash Card application developed in Python using the Tkinter GUI library.

This project was created as part of the **100 Days of Code – The Complete Python Pro Bootcamp** by Dr. Angela Yu.

---

## Features

- Random French vocabulary flash cards
- Automatic card flip to reveal the English translation
- Track learned words
- Save remaining words to a CSV file
- Simple graphical user interface

---

## Technologies

- Python
- Tkinter
- Pandas
